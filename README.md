# Usage


Open chisel with `:Chisel`


# Installation


```lua
use {
    "TheSnakeWitcher/chisel.nvim,
    requires = "akinsho/toggleterm.nvim",
}
```


# Configuration


To configure call function `setup`. 

```lua
require('chisel').setup({
    direction = "float"   -- others options are `vertical` and `horizontal` 
})
```
